BOM(Browser Object Model)
=======================
> 웹 브라우저의 창 또는 프레임을 추상화해서 프로그래밍적으로 제어할 수 있도록 제공하는 수단

# 1. Window
> 인용
## 1.1 전역객체 Window
### 1.1.1. 메소드
```
window.alert();
alert();
같은 표현 
window는 생략 가능하다.
```
### 1.1.2. 변수
```
var a = 1;
alert(window.a);
alert(a);
같은 표현
```
### 1.1.3. 객체
```
var b = { id:1 };
alert(window.b.id);
alert(b.id);
```
