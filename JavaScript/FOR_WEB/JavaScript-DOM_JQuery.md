DOM_JQuery
=======================
> JQuery는 DOM을 내부에 감추고 보다 쉽게 웹페이지를 조작 할 수 있도록 돕는 도구 (라이브러리)  
**사용법**  
구조
```
$('선택자').메소드('속성', '값');
$('선택자').메소드({'속성' : '값'});

```
기본
```
    JQuery(document).ready(function($){
      $('선택자').메소드();
})
```
요약(간략)   
```
    JQuery(document).ready(function($){
      $('선택자').메소드();
})
```
기본 버전은 이벤트를 이용한 방법으로 HTML 문서의 준비가 끝나면 실행된다.
